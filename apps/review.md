# レビューAppの対応

商品ページにレビュー情報を表示できるAppsです。

## 関連するテンプレート変数

### if Apps

| 変数 | 説明 |
|-----|-----|
| {block:AppsReview} | レビューAppsをインストールしている |

### Apps

| 変数 | 説明 |
|-----|-----|
| {AppsReviewTag} | レビューのタグ。商品ページでのみ表示される。 (レビューAppのインストールが必要) |

## レビューの表示

商品ページにレビューAppの変数を追加します。

*例)*

```
{block:AppsSearch}
{AppsReviewTag}
{/block:AppsSearch}
```

## 動作のチェック
- 商品ページでレビューが表示されるか
